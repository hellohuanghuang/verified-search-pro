# 行为准则

Verified Search Pro 欢迎所有背景的贡献者。我们承诺提供一个友好、安全和包容的社区环境。

## 我们的承诺

- 尊重每一位参与者，无论其经验水平、性别、身份、国籍或观点。
- 使用建设性语言，避免人身攻击、歧视或骚扰。
- 耐心对待新手问题，鼓励学习和成长。
- 专注于技术和产品改进，而非无意义的争论。

## 不可接受的行为

- 侮辱、贬损或敌对言论。
- 骚扰、恐吓或持续性不当关注。
- 歧视性、仇恨性或冒犯性内容。
- 故意破坏项目秩序或他人工作成果。

## 如何报告

如果你遇到违反行为准则的情况，请通过 GitHub issue 或邮件联系维护者。

## 适用范围

本行为准则适用于所有项目空间，包括 issue、PR、讨论区和任何相关活动。
