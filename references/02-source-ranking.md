# 信息源五级分类与使用规则

## 五级分类

| 等级 | 来源类型 | 使用规则 | 置信度 | 示例 |
|------|---------|---------|--------|------|
| **A** | 政府官网、权威媒体、学术期刊、品牌/机构官方 | 优先使用，可直接引用 | **高** | gov.cn、reuters.com、nature.com |
| **B** | 知名学者/评论员/媒体人的署名内容 | 可用，需确认作者身份 | **中高** | 知乎认证号、知名公众号、专家专栏 |
| **C** | 一般 UGC（知乎高赞、一般公众号、行业论坛） | 仅作观点参考，必须交叉验证 | **中** | 知乎普通回答、行业论坛帖子 |
| **D** | 百度百科、未署名自媒体、内容农场 | 极度谨慎，仅用于基础概念 | **低** | baike.baidu.com、未署名自媒体 |
| **E** | 匿名论坛、营销号、无来源截图 | 不使用 | **极低** | 匿名论坛帖子、营销号文章 |

## 特殊规则

### 知乎（zhihu.com）
- 不一刀切排除
- 认证作者/高赞回答 → B 级（需确认作者身份）
- 普通回答 → C 级（仅观点参考）
- 匿名回答 → D 级

### 微信公众号（mp.weixin.qq.com）
- 知名媒体/机构号 → B 级
- 个人认证号（有专业背景）→ B-C 级
- 普通号 → C 级
- 未认证/营销号 → D 级

### 百度百科（baike.baidu.com）
- 仅用于基础概念定义
- 永远不可用于事实、数据、争议性内容
- 引用时必须标注"据百度百科"

**兜底规则**：
- 不在表中的域名 → 默认 `unknown`，评分 0.5
- `.gov` / `.gov.cn` → 自动 A 级，评分 0.9
- `.edu` / `.ac.cn` → 自动 A 级，评分 0.85
- 其他未分类域名 → 按内容质量动态评估

## 降级规则

- A 级来源缺失时，可用 B 级替代，但需标注
- B 级来源有矛盾时，降级到 C 级处理
- 同一信息仅出现在 C 级来源 → 标注"单一来源，待验证"
- 信息仅出现在 D-E 级来源 → 不使用
